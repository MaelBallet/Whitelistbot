# Whitelist Bot

Setup your .env file

	# .env
	DISCORD_TOKEN={your-bot-token}

## ***2 Options:***

- #### *Option 1*
Download git then execute -> 

	pip install -r requirements.txt

- #### *Option 2* 
Create virtual env using 

	python -m venv .venv

Activate virtual env 

	cd .venv/scripts
	activate.bat

Then install requirements in venv using

	pip install -r requirements.txt


## ***Start the bot***

Execute bot.py

	python bot.py
